# 성능 마케터를 위한 Adobe GenStudio Beta 릴리스 노트

이 노트에서는 9월 19일로 끝나는 주에 대한 성능 마케터의 수정 사항 및 개선 사항에 대한 중요한 Adobe GenStudio을 강조합니다.

## 새로운 기능 및 향상된 기능

* **Adobe Experience Manager Assets과 통합**. 이제 Adobe Experience Manager Assets에 대한 읽기 전용 액세스를 사용할 수 있습니다. <!-- GS-2432 -->

* **템플릿 업데이트 워크플로우에 대한 개선 사항**.  이제 템플릿을 업데이트하는 사용자가 템플릿을 사용할 채널을 선택합니다. <!-- GS-4029 -->

* **페이지 로드 만들기 성능이 향상되었습니다**. 사용되지 않는 종속성이 페이지 로드 프로세스에서 제거되었습니다. <!-- GS-3630 -->

* **여러 섹션 전자 메일 지원**. 이제 편집자는 여러 섹션 및 이미지가 포함된 이메일을 생성할 수 있습니다. 생성된 이메일의 특정 조각(예: 헤드라인)을 다시 생성할 수도 있습니다. <!-- GS-2436 -->

* **만드는 동안 데스크톱 보기와 모바일 보기 간을 전환합니다**. 이제 사용자는 데스크탑 보기와 모바일 보기 간을 전환하여 이메일 경험 변형을 미리 볼 수 있습니다. <!-- GS-4314 -->

* 이제 콘텐츠는 원래 에셋 차원에 상대적인 자르기 차원으로 이미지를 생성합니다. <!-- GS-3150 -->

* 이제 사용자는 생성된 이미지 변형을 선택하고 자르기 조정 기능을 사용하여 생성 워크플로 중에 자를 수 있습니다. <!-- GS-5538 2342 -->

* 이제 승인된 경험의 세부 정보 보기에 해당 경험에서 참조되는 모든 에셋의 상태와 썸네일 이미지가 표시됩니다. <!-- GS-3783 -->

## 알려진 문제

다음 알려진 문제는 GenStudio for Performance Marketers GA 릴리스에서 해결될 예정입니다.

* 일부 [!DNL Create] 캔버스 작업에 일시적인 대기 시간 문제가 영향을 미칩니다. <!-- GS-5203 -->

* 이메일 생성으로 인해 이메일이 불완전합니다. **해결 방법**: 페이지를 새로 고치고 다시 생성합니다. <!-- GS-5209 -->

* 템플릿을 업로드할 수 있지만 볼 수 없습니다. **해결 방법**: **[!UICONTROL 캠페인]** 필드가 채워진 에셋을 업로드합니다. 그런 다음 템플릿을 다시 업로드합니다. <!-- GS-4815 -->

* facebook에 로그인할 때 채널 메타 광고 계정에 두 번 로그인해야 합니다. **해결 방법**: 채널 MetaAds 계정에 로그인하기 전에 Facebook에서 로그아웃합니다. <!-- GS-4806 -->

### 추가 개선 사항 및 해결된 문제

* 이제 끌어서 놓기가 프롬프트 영역에서 예상대로 작동합니다. <!-- GS-3977 -->

* Tab 키를 사용하여 왼쪽 탐색 막대의 요소를 탐색할 때 발생하는 문제를 해결했습니다. 이전에는 한 요소에서 다음 활성 요소로 탐색하려면 여러 번의 클릭이 필요했습니다.  <!-- GS-2639 -->

* 이제 GenStudio은 경험이 로드되는 동안 사용자가 이름을 편집할 때 경험 이름을 저장합니다. <!-- GS-5242 -->

* 이제 사용자가 경험 제목을 성공적으로 편집할 수 있습니다. 이전에는 제목 텍스트를 편집하려고 하면 원래 텍스트가 기본값으로 사용되었습니다. <!-- GS-5246 -->
* 이제 선택한 이미지가 다중 부분 전자 메일 만들기 중에 예상대로 캔버스에 렌더링됩니다. <!-- GS-5263 -->

* 이제 [!DNL Content] 경험 세부 정보 페이지의 모든 문자열이 현지화되었습니다. <!-- GS-5016 -->

* 이제 사용자는 [!DNL Products]에서 세부 정보 보기가 열려 있는 제품을 삭제할 수 있습니다. <!-- GS-5057 -->

* 검색에서 일치하는 결과가 나오지 않을 때 GenStudio에서 표시하는 메시지가 개선되었습니다. <!-- GS-4544 -->

* 검색 필터 값에 대한 `aria-label` 특성 값이 예상대로 변환되었습니다. <!-- GS-5388 -->

* 이제 사용자는 [!DNL Create] 캔버스 프롬프트 영역에서 중복 에셋을 삭제할 수 있습니다.  <!-- GS-5233 -->

* 이제 계정 필터가 경험, 에셋 및 특성에서 예상대로 작동합니다. <!-- GS-4812 -->

* 가독성과 접근성을 개선하기 위해 메타 광고 템플릿의 글꼴 문제가 해결되었습니다. <!-- GS-5354 -->

* 이제 초안 제목이 예상대로 변경됩니다. 이전에는 제목을 편집한 후 기본 이름으로 되돌렸습니다. <!-- GS-2951 -->

#### 템플릿 수정 사항

* 이제 크기 조정 기능이 메타 광고 템플릿의 여러 이미지에서 예상대로 작동합니다. 이전에는 GenStudio에서 선택한 모든 템플릿에 대한 이미지 크기를 조정하지 않았습니다. <!-- GS-4696 -->

* 이제 템플릿을 삭제하면 [!DNL Content] 페이지가 예상대로 새로 고쳐집니다. <!-- GS-5397 -->

* 이제 드롭다운 메뉴에서 [!DNL Personas], [!DNL Brands] 또는 [!DNL Products]의 값만 선택할 수 있습니다. 이전에는 _템플릿 업로드_ 대화 상자를 통해 사용자가 [!DNL Persona], [!DNL Brand] 또는 [!DNL Product] 이름을 입력할 수 있었습니다. <!-- GS-5072 5071-->

* 이제 템플릿 업로드 프로세스 중에 **[!UICONTROL 뒤로]** 단추를 사용할 수 없습니다. <!-- GS-5358 -->

* [!DNL Create] 템플릿 선택 세부 사항 보기의 모든 문자열이 현지화되었습니다. <!-- GS-5025 -->

## 이전 Beta 릴리스

이전 Beta 릴리스에는 다음과 같은 특징 및 수정 사항이 포함되었습니다.

### 강조 표시

* 자산 로드를 개선하기 위해 [!DNL Create] 콘텐츠 선택기가 리팩터링되었습니다. <!-- GS-2586 -->

* 이제 GenStudio에서 [!DNL Insights] 테이블 및 갤러리 보기에서 미디어 에셋을 미리 보는 옵션을 지원합니다. 비디오 축소판에는 음소거 옵션이 있는 **재생** 단추가 포함되어 있습니다. <!-- GS-4398 -->

* Instagram 및 Facebook 채널 지침이 메타 브랜드 지침으로 결합되었습니다.

* [!DNL Create] 캔버스 탐색 요소가 간소화되었습니다. [!DNL Create] 랜딩 페이지에 왼쪽 탐색 패널이 표시되지만, 이제 사용자는 **[!UICONTROL 뒤로]** 단추를 사용하여 다른 [!DNL Create] 작업 영역에서 이 스페이스로 이동합니다.

* 탐색 요소가 다음과 같은 제품 영역을 포함하여 제품 전체에서 작업을 수행하는 동안 사용자 포커스를 지원하도록 개선되었습니다.

   * [!DNL Content]의 자산, 경험, 템플릿 세부 정보
   * [!DNL Insights]의 경험, 자산, 특성 세부 정보
   * [!DNL Brands]의 브랜드 세부 정보
   * 제품 및 담당자의 제품 및 담당자 세부 정보

* 사용자는 [!DNL Content]에서 경험에 대한 업데이트를 보려면 더 이상 **[!UICONTROL 새로 고침]** 단추를 클릭할 필요가 없습니다.

* 이제 _경험 세부 정보_ 페이지에서 외부 에셋 썸네일을 HTML으로 렌더링합니다.

* Assets 및 경험 추가 또는 삭제 후 UI 지연 시간이 개선되었습니다.

* 이제 템플릿 미리보기에 더 설명적인 기본 텍스트가 포함됩니다. [템플릿 사용자 지정](https://experienceleague.adobe.com/en/docs/genstudio/user-guide/content/templates/customize-template#template-preview)을 참조하세요.

* **백분율 기반 유효성 검사 점수**: 이제 브랜드 유효성 검사는 브랜드 유효성 검사 점수를 통과/실패 값이 아닌 백분율로 표시합니다. (8/16 고정)

* **업데이트된 브랜드 추출 인터페이스**: 이제 브랜드 추출에 추출 프로세스 완료가 백분율로 표시됩니다. (8/16 고정)

* **추출 중 증분 브랜드 로드**: 이제 브랜드 가이드라인이 사용자 인터페이스에 점진적으로 로드됩니다. (8/16 고정)

* **여러 섹션 전자 메일 만들기**: 이제 사용자는 별도의 헤드라인, 이미지, 본문 및 CTA 요소로 구성된 전자 메일을 만들 수 있습니다. (8/16 고정)

* **메타데이터 크기 조정**: 편집자는 메타데이터 종횡비의 크기를 조정할 수 있습니다. (8/16 고정)

* **제한된 [!DNL Insights]개의 로그인 계정**: 이제 [!DNL Insights] 로그인은 고객당 하나의 계정만 지원합니다. (8/16 고정)

### 추가 개선 사항 및 해결된 문제

* 이제 [!DNL Create] 캔버스가 메타 광고의 이미지를 올바르게 렌더링합니다. (9/13 수정) <!-- GS-4864 -->

* 메타데이터 캔버스 미리 보기와 내보낸 보기 간에 불일치가 있을 수 있지만 내보낸 경험은 예상대로 작동합니다. (9/13 수정) <!-- GS-4492 4401 -->

* 이제 업로드된 이미지에 예상 스마트 태그가 포함됩니다. (9/13 수정) <!-- GS-4856 -->

* 이제 메타 광고 내보내기 CSV 파일에 예상대로 이미지가 포함됩니다. 이전에는 ZIP 파일에 이미지 대신 CSV 내보내기 파일과 NULL 파일이 포함되었습니다. (9/13 수정) <!-- GS-5107 -->

* 이제 예상대로 템플릿 세부 정보 보기 **[!UICONTROL 업로드한 사람]** 필드에 텍스트를 입력할 수 있습니다. 이전에는 로딩 아이콘으로 인해 사용자가 텍스트를 입력할 수 없었습니다. (9/13 수정) <!-- GS-4887 -->

* 브랜드가 삭제된 사용자는 더 이상 브랜드 세부 사항 보기로 리디렉션되지 않습니다. (9/13 수정) <!-- GS-2663 -->

* 편집자는 검토 및 승인을 위해 변형을 보낼 때 더 이상 다음 오류를 받지 않습니다. `You have no access to view comments on this Object`. (9/13 수정) <!-- GS-5140 -->

* 검토 및 승인 워크플로에서 사용하는 이메일 템플릿이 업데이트되었습니다. (9/13 수정) <!-- GS-5239 -->

* 이제 템플릿 선택기 로드 중에 네트워크 오류가 발생하면 GenStudio에 오류 메시지가 표시됩니다. (9/13 수정) <!-- GS-4682 -->

* 에셋, 경험 또는 템플릿 카드에서 선택한 개체로 이동할 때 발생하는 문제가 해결되었습니다. (9/13 수정) <!-- GS-4390 -->

* 이제 [캔버스 만들기]에서 열면 _Assets 추가_ 팝업이 지역화됩니다. (9/13 수정) <!-- GS-4867 -->

* 이제 브랜드 유효성 검사가 다시 생성된 변형에 대해 트리거됩니다. 이전에는 편집기가 기존 초안의 변형을 재생성한 경우 유효성 검사가 트리거되지 않았습니다. (9/13 수정) <!-- GS-3971 -->

* _Assets 추가_ 팝업이 예상대로 현지화되었습니다. (9/5 수정) <!-- GS-3834 -->

* 메타 광고 경험 템플릿 크기 조정 문제가 해결되었습니다. (9/5 수정) <!-- GS-4174 -->

* 다중 파트 이메일용 CSV 내보내기 파일의 텍스트 필드가 이제 예상대로 순서가 지정됩니다. (9/5 수정) <!-- GS-4013 -->

* 사용자가 **백스페이스** 키를 반복하여 검색 필드 텍스트를 지우면 [!DNL Content] 검색 필드가 더 이상 사라지지 않습니다. (9/5 수정) <!-- GS-4543 -->

* 이제 공동 작업자가 댓글에 `@` 언급을 추가하면 GenStudio for Performance Marketters가 예상대로 사용자를 로드합니다. 이전에는 사용자가 로드되지 않았으며 오류가 표시되었습니다. `Unable to load users. Refresh the page`. (8/29 수정) <!-- GS-4113 -->

* GenStudio은 프롬프트 영역에서 전자 메일을 만드는 동안 편집기가 **콘텐츠 선택**&#x200B;을 클릭할 때 **문제가 발생했습니다** 메시지를 더 이상 표시하지 않습니다. <!-- GS-4879 -->

* 이제 _경험 세부 정보_ 페이지 피드 배치 이름이 Facebook 또는 Instagram 피드를 지정합니다. (8/16 고정)

* 사용자가 _자산 개요_ 보기에서 _자산 세부 정보_ 보기로 이동할 때 더 큰 이미지와 비디오를 자르기가 일관됩니다. (8/16 고정)

* 사용자가 로그인하기 전에 특성 화면 검색 결과 수에 더 이상 `0 of`이(가) 표시되지 않습니다. (8/16 수정) <!-- GS-3665 -->

* **[!UICONTROL [!DNL Insights]]** > **[!UICONTROL 자산]** 개수 필드를 클릭하면 더 이상 검색 및 필터 설정이 지워지지 않습니다. (8/16 수정) <!-- GS-3476 -->

* 사용자가 [!DNL Insights] 보기에서 자격 증명을 입력하려고 하면 GenStudio에 오류가 표시됩니다. (8/29 수정) <!-- GS-4689 -->

* ACP 스토리지 플랫폼 문제로 인해 브랜드 지침을 업로드할 수 없습니다. (8/22 수정) <!-- GS-4369 -->

* 전자 메일을 만드는 동안 프롬프트 영역 [!DNL Brands] 드롭다운 메뉴에 [!DNL Brands] 목록의 끝에 회전기가 표시됩니다. (8/22 수정) <!-- GS-4077 -->
